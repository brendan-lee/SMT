    前言

    内容详解原表大致框架及翻译由百度贴吧-大王派提供，游戏1.22.7及以后更新内容和其他更多解释说明由百度贴吧-销锋镝铸编写。
    目前本页内容对应游戏1.29。

    第一节 数据类型

    ——每个属性能够支持的数据类型
    英文名 中文名 数值范围
    Boolean 布尔值 [TRUE/FALSE]
    int 4字节整数 [-2147483648 ~ 2147483647]
    uint 4字节非负整数 [0 ~ 4294967295]
    double 双精度浮点数 [-10e+17 ~ +10e+308]
    String 字符串 不超过256字符的任意内容
    Array 数组 逗号分隔的一组数字
    特殊值：Infinity 无穷大[仅限double类型]

    第二节 内容详解

    ——每一行第n个分号之后所代表的内容详解
    n 数据类型 英文原名 中文翻译 说明
    0 String Class Name 类名 游戏内部文件使用的名字，例如物品被破坏和动物死亡掉落的物品就要用到
    1 String Default Display Name 默认显示名称 游戏中玩家看到的物品名字
    2 String Default Category 默认类别 帮助中的类别
    3 String Behaviors 行为 即特殊属性多种行为可以用逗号隔开，详情见附录
    4 uint Display Order 显示顺序 游戏中创造模式物品栏中物品的显示顺序(同时是这个方块在该文件中所在的行数)
    5 Array(2) Icon Block Offset 图标偏移 x,y方向
    6 Array(3) Icon View Offset 图标偏移
    7 uint Default Icon View Scale 默认图标视角大小缩放 在物品栏里显示的大小
    8 uint First Person Scale 第一人称大小缩放 第一人称视角下手持物品的大小缩放倍数
    9 uint First Person Offset 第一人称位置偏移 第一人称视角下手持物品的位置偏移
    10 uint First Person Rotation 第一人称物品旋转 第一人称视角下手持物品的旋转角度
    11 uint In Hand Scale 在手里的大小缩放 第三人称视角下手持物品的大小缩放倍数
    12 Array(2) In Hand Offset 在手里的位置偏移 第三人称视角下手持物品的位置偏移
    13 uint In Hand Rotation 在手里时旋转角度 第三人称视角下手持物品的旋转角度
    14 String CraftingId 合成标识 在CraftingRecipes.xml中作为合成材料时方块标识
    15 int Default Creative Data 默认创造模式是否提供 -1不提供，0创造模式免费提供
    16 Boolean Is Collidable 是否可碰撞 当此项为FALSE时，主角和动物可以穿过它(例如草和花)；TRUE时则不可穿过（例如石头和土）
    17 Boolean Is Placeable 是否可放置 工具武器什么的改成TRUE便可以放置到地上，但不可见
    18 Boolean Is Digging Transparent 挖掘时是否透明 是否可挖掘
    19 Boolean Is Placement Transparent 是否放置透明 是否可以在它上面放东西，如果是TRUE就会点到后面的方块来放置
    20 Boolean Is Interactive 是否交互 电器此项为TRUE，但改了没任何效果
    21 Boolean Is Editable 是否可编辑 内存条，真值表此值为TRUE
    22 Boolean Is Gatherable 是否可收集 即可不可以捡起来
    23 Boolean Has Collision Behavior 是否有碰撞动作 压力板此属性为true，改其他物品貌似无效果
    24 Boolean Kills When Stuck 卡住时杀死卡在其中的实体 仙人掌和倒刺板此值为TRUE
    25 Boolean Is Fluid Blocker 能否阻挡流体 流体指水和岩浆，空气此属性为TRUE时它们就不会乱流了
    26 Boolean Is Transparent 是否透明 即是否实体，是就代表材质透明时黑的，否就代表透明材质时透明。同时决定是否被沙子压碎。还和方块刷新位置有关，如果玄武岩上方会长生的方块此属性为”TRUE”时就会刷新到地面（直接在草地上）
    27 uint Shadow Strength 阴影强度（-1~15） -1就是实体方块，方块透明的同时还必须把它改成大于等于0
    28 uint Light Attenuation 光衰减 仅透明方块有效，光透过方块时衰减的亮度，越小越透光
    29 uint Emitted Light Amount 发光强度 9时照亮一格，15时照亮7格，以上虽然照亮更大范围，但会出bug，最大23
    30 uint Object Shadow Strength 物体阴影强度
    31 String Default Drop Content 默认掉落物品 写#号开头的要掉落的物品的标识符。花，稻子没法改，若掉落自己本身，直接写个#就行
    32 uint Default Drop Count 默认掉落数量
    33 uint Required Tool Level 需求工具等级 挖掘时需求的装备等级，钻石矿为4
    34 uint Max Stacking 最大背包 箱子,工作台和火炉里可以堆叠的数量
    35 uint Max In Hand Stacking 最大手持 下面6个方框堆叠数量
    36 Boolean Sleep Suitability 适合睡觉 0.5好像是最适合，0代表不能睡
    37 double Friction Factor （行走）摩擦系数 越小越滑，为0时无法走动
    38 double Density 密度 物品掉落水中浮沉的指标
    39 Boolean No Auto Jump 走上方块不执行自动跳跃 此值为TRUE时面前遇到一格高此方块将自动停住或绕道
    40 Boolean No Smooth Rise 进入其中不执行流畅上升 其中栅栏此属性为TRUE所以动物不会跳过去
    41 uint Fuel Heat Level 燃料等级 只有煤的此属性是2, 烧铁块需求的等级是2
    42 uint Fuel Fire Duration 燃料燃烧持续时间 相当于热值,越大烧的时间越长，产出越多
    43 String Sound Material Name 声音材料名称 对应到Content/Audio里的文件
    44 String Digging Sounds Folder 挖掘声音文件夹 被挖掘时发出的声音所在的位置
    45 double Digging Sounds Volume 挖掘声音音量
    46 double Destruction Sounds Volume 破坏声音音量
    47 double Shovel Power 铲子效率 各种工具（ShovelPower等）是挖掘速度（每秒）（不含反应/识别时间）例如Quarry Power是20的工具挖Dig Resilience是60的方块要3秒空手的速度目前测得全部为1
    48 double Quarry Power 镐子效率
    49 double Hack Power 斧头效率
    50 double Default Melee Power 打击(近战)攻击力
    51 double Default Melee Hit Probability 默认近战击中概率 在1.27某些版本中，它处于DefaultMeleePower与DefaultProjectilePower之间，现已消失
    52 double Default Projectile Power 投掷攻击力 丢出去的物品的攻击力
    53 uint Tool Level 工具等级 木头工具该属性为1，石头为2，铁为3，钻石为4
    54 uint Durability 工具耐久度 即使用次数，留空或-1将无耐久度
    55 Boolean Is Aimable 是否可瞄准 可投掷物品及弓此属性都为TRUE
    56 Boolean Is Stickable 击中方块是否停留 比如矛的为TRUE，煤，铁锭为FALSE。若该方块有BombBlockBehavior则按一定概率穿过Projectile Stick Probability大于0的方块
    57 Boolean Align To Velocity 成直线速率 此值为TRUE时该物品被抛出看起来是横向的(例如矛)，不建议给无3D模型的物品调成TRUE（实在不知如何翻译）
    58 double Projectile Speed 投掷速度 亲测就是最大投掷距离
    59 double Projectile Damping 投掷阻力(衰减) 改为零出现超时空现象
    60 Array(3) Projectile Tip Offset 扔出点偏移
    61 Boolean Disintegrates On Hit 撞击时碎裂 蛋的此属性为TRUE
    62 double Projectile Stick Probability 物品粘性可能性 使IsStickable为TRUE的方块被投掷到它上面粘住的可能性(若那个物品有BombBlockBehavior属性则为穿过可能性)
    63 double Fire Duration 露天点火持续时间 这个值改成小数会向下取整
    64 double Explosion Resilience 爆炸抗性 爆炸血量，越大越耐炸，可以为无限
    65 double Default Explosion Pressure 默认爆炸压力 此值影响爆炸范围和威力。对范围的影响是要多少倍范围乘1后面多少个0，例如2倍范围数值需要乘100。威力和此值成正比，为每秒的伤害
    66 uint IsExplosionTransparent 是否爆炸透明 此值为TRUE时该物品可被炸飞
    67 String Dig Method 挖掘方法 取消挖掘透明这里至少要选一种工具
    68 double Dig Resilience 挖掘抗性 越大要挖得越久，基岩这个属性是无限
    69 double Projectile Resilience 弹射物抗性 越大越不容易被矛等投掷出的物品砸破
    70 double Nutritional Value 营养价值 大于0时可以食用，范围0-1，食用增加等量饱食度
    71 uint Food Type 食物类型 可自定义，Database.xml中规定了每种动物对不同食物类型的喜好程度，原版有Fruit、Grass、Meat、Fish
    72 uint Default Texture Slot 默认材质位置 材质图上从左向右，从上向下排列顺序，第一个（草地顶面）是0，第二个（花岗岩）是1
    73 double Destruction Debris Scale 破坏碎片大小缩放 方块被破坏时爆出的一堆碎片一样的东西的大小缩放倍数
    74 String Default Description 默认描述 会在游戏帮助的方块说明界面出现，可以随意删除添加内容

    第三节 启发例子

    (提示：善于使用查找替换功能以及正则表达式，方便批量修改。)

    1.物体发光

    这里的物品必须是可以放置的，船和矛什么的也不行
    修改要修改的方块的Emitted Light Amount为9~22,Light Attenuation为大于0（一般改为1）

    2.狙击矛

    修改要修改的可投掷方块的Projectile Speed为较大值(例如9999)，Projectile Damping为0(会导致超时空现象，即瞬间击中目标并弹回到主角面前，也可不改)

    3.铁砂掌

    (生存模式像创造模式一样破坏物品)
    修改所有方块的Dig Resilience为0

    4.超级工具

    即秒砍挖铲杀，耐久几乎无限。
    修改要修改的工具方块的ShovelPower，QuarryPower，HackPower为99，DefaultMeleePower或DefaultProjectilePower为999(虎鲸攻击抗性为200)，ToolLevel为4，ToolDurability为50000(没人用的完，比无限省3个字节，且太大容易出错)

    5.超级大炮

    修改任意火药桶方块的Default Explosion Pressure为99999，作镗的方块的Explosion Resilience为Infinity

    6.会掉落物品的水，岩浆，火

    修改WaterBlock、MagmaBlock、FireBlock的Digging Transparent为FALSE，Default Drop Content为你想要让它掉落的方块，Dig Method若原来为空，则需要改为一种工具类别，包括Shovel、Quarry、Hack

    7.基岩综合

    （基岩可挖掘，拾取，创造模式提供）
    修改BedrockBlock的Is Placeable、Is Gatherable为TRUE，Default Creative Data、Dig Resilience为0

    8.水透视

    (通过水看地下岩浆分布，也会带来副作用，大海表面被草原覆盖)
    修改WaterBlock的Is Transparent为FALSE，Light Attenuation为0，Emitted Light Amount为适当值使方块发光

    9.超级挖沙挖落岩

    (挖一块连着它的方块全部破坏)
    修改SandBlock和GraveBlock的Is Transparent为TRUE，Shadow Strength为大于等于0的数。

    10.快速除水

    （引爆大炸弹将一定范围内流体（水和岩浆）全部炸掉，同时大炸弹可以直接放置于水中并在水中爆炸，而其他方块不受影响；空气能阻止流体继续流动）
    修改WaterBlock和MagmaBlock的ExplosionResilience为0，IsExplosionTransparent为True，修改WaterBlock的IsPlacementTransparent为False，修改LargeGunpowderKegBlock的IsFluidBlocker为True，DefaultExplosionPressure为99999，修改除WaterBlock和MagmaBlock外所有方块的ExplosionResilience为Infinity

    附录

    注：本附录最后一次更新对应的游戏版本为1.25左右

    (1)此项输入以下内容此方块将获得各种不同特殊属性，原来留空但有特殊属性的不会发生覆盖，而会同时拥有这两种特殊属性(收集于Database.xml)

    • FertilizerBlockBehavior此方块可作为肥料使用，同时将导致不可放置。发现：TallGrassBlock有此属性时，地形生成时该方块下方的土块会一同消失
    • FireBlockBehavior此方块将自燃并消失，并引燃其他可燃物，也可直接点可燃物上表面点燃该物品
    • MatchBlockBehavior此方块成为火柴，可点燃可燃物，将不可放置
    • RakeBlockBehavior此方块可作为耙子使用
    • SaplingBlockBehavior此方块放置到土地上如果条件适合，将长成Oak树，若条件不允许将直接消失。用途示例：让杂草长成树木
    • CactusBlockBehavior此方块将从上面长出仙人掌，如果此方块不是沙子，长出的仙人掌将直接破坏并掉落，此方块也将只能放置在沙子和仙人掌上表面。用途示例：长满仙人掌沙漠，沙滩；刷仙人掌机
    • IvyBlockBehavior此方块若透明将只能挂在树上(未知原因经常挂不上)，并向下长出此方块。用途示例：刷物品机
    • TreasureGeneratorBlockBehavior此方块将在周围有方块变为空气时随机掉落宝物。
    • SaddleBlockBehavior此方块将可作为马鞍使用
    • WhistleBlockBehavior此方块将可作为哨子使用，并将不可放置
    • ThrowableBlockBehavior此方块将可投掷，需要调整其他和投掷有关属性
    • MagnetBlockBehavior此方块被放置后将相当于一个磁铁
    • ExplosivesBlockBehavior此方块将可爆炸，需要调整其他和爆炸有关属性。
    • MagmaBlockBehavior此方块将点燃周围可燃物如果该物品可燃则自己也会燃烧。不一定对所有方块有效
    • GrassBlockBehavior此方块上表面放置实心方块后变成DirtBlock
    • SoilBlockBehavior此方块上表面放置实心方块后变成DirtBlock
    • BombBlockBehavior此方块投掷出后可爆炸，同时也会影响Is Stickable
    • BowBlockBehavior有此属性后该方块在物品栏放一个弓箭到上面会变成弓

    (2)一些有效果但不知道如何有那种方块原本拥有的特殊属性的属性：

    • GraveBlockBehavior将可导致方块只能放到一个方块上表面，下方方块被破坏后上面的此方块将破坏(原版GraveBlock会向下掉落)。用途示例：撸树mod

    • BatteryBlockBehavior

    • EggBlockBehavior
